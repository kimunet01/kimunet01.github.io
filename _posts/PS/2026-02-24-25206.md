❗ 백준 25206번 문제 : `너의 평점은`  
```
인하대학교 컴퓨터공학과를 졸업하기 위해서는, 전공평점이 3.3 이상이거나 졸업고사를 통과해야 한다.
그런데 아뿔싸, 치훈이는 깜빡하고 졸업고사를 응시하지 않았다는 사실을 깨달았다!

치훈이의 전공평점을 계산해주는 프로그램을 작성해보자.

전공평점은 전공과목별 (학점 × 과목평점)의 합을 학점의 총합으로 나눈 값이다.

인하대학교 컴퓨터공학과의 등급에 따른 과목평점은 다음 표와 같다.

A+	4.5
A0	4.0
B+	3.5
B0	3.0
C+	2.5
C0	2.0
D+	1.5
D0	1.0
F	0.0
P/F 과목의 경우 등급이 P또는 F로 표시되는데, 등급이 P인 과목은 계산에서 제외해야 한다.

과연 치훈이는 무사히 졸업할 수 있을까?
```

💫 `접근 방법`
```
입력 형식이 정해져 있으므로 이름, 학점, 성적 에 해당되는 값을 입력받음.
20줄에 걸쳐 입력되므로 반복문은 20회.

학점 * 과목평점을 학점의 합으로 나누면 되기 때문에
입력을 받을때마다 학점의 총합, 성적의 총합을 따로 연산.
최종 결과에 성적의 총합 / 학점의 총합을 하면 완성.
```

⭕ `코드`
```c++
#include <iostream>
using namespace std;

int main() {
	string subject;
	double point;
	string score;

	double grade{ 0 };
	double total_point{ 0 };

	for (int i = 0; i < 20; ++i) {
		cin >> subject >> point >> score;
		if (score == "A+") {
			grade += point * 4.5;
			total_point += point;
		}
		else if (score == "A0") {
			grade += point * 4.0;
			total_point += point;
		}
		else if (score == "B+") {
			grade += point * 3.5;
			total_point += point;
		}
		else if (score == "B0") {
			grade += point * 3.0;
			total_point += point;
		}
		else if (score == "C+") {
			grade += point * 2.5;
			total_point += point;
		}
		else if (score == "C0") {
			grade += point * 2.0;
			total_point += point;
		}
		else if (score == "D+") {
			grade += point * 1.5;
			total_point += point;
		}
		else if (score == "D0") {
			grade += point * 1;
			total_point += point;
		}
		else if (score == "F") {
			grade += point * 0;
			total_point += point;
		}
		else if (score == "P") {
			continue;
		}
	}
	grade = grade / total_point;
	std::cout << grade;
}
```
