❗ 백준 2566번 문제 : `최대값`  
```
9×9 격자판에 쓰여진 81개의 자연수 또는 0이 주어질 때,
이들 중 최댓값을 찾고 그 최댓값이 몇 행 몇 열에 위치한 수인지 구하는 프로그램을 작성하시오.
```

💫 `접근 방법`
```
행렬의 크기가 고정되어 있으므로 이차원 행렬을 선언 후에
각 값을 대입하면서 최대값도 수정하는 방식.
```

⭕ `코드`
```c++
#include <iostream>
using namespace std;

int main() {
	int arr[9][9];
	int max{ 0 };
	int max_row{ 0 };
	int max_col{ 0 };

	for (int i = 0; i < 9; ++i) {
		for (int j = 0; j < 9; ++j) {
			int tmp;
			cin >> tmp;
			arr[i][j] = tmp;
			if (tmp > max) {
				max = tmp;
				max_row = i;
				max_col = j;
			}
		}
	}
	cout << max << '\n' << max_row+1 << ' ' << max_col+1;
}
```
