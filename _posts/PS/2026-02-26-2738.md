❗ 백준 2738번 문제 : `행렬 덧셈`  
```
N*M크기의 두 행렬 A와 B가 주어졌을 때, 두 행렬을 더하는 프로그램을 작성하시오.
```

💫 `접근 방법`
```
이중 포인터로 먼저 공간을 확보 한 후에
데이터를 입력받고
두번째 입력 데이터를 받을때 원본 배열에 덧셈을 해주는 방식.
```

⭕ `코드`
```c++
#include <iostream>
using namespace std;

int main() {
	int n, m;
	cin >> n >> m;
	int** arr = new int*[n];
	for (int i = 0; i < n; ++i) {
		arr[i] = new int[m];
	}
	for (int i = 0; i < n; ++i) {
		for (int j = 0; j < m; ++j) {
			cin >> arr[i][j];
		}
	}
	for (int i = 0; i < n; ++i) {
		for (int j = 0; j < m; ++j) {
			int a;
			cin >> a;
			arr[i][j] += a;
		}
	}
	for (int i = 0; i < n; ++i) {
		for (int j = 0; j < m; ++j) {
			cout << arr[i][j] << ' ';
		}
		cout << '\n';
	}
	for (int i = 0; i < n; ++i) {
		delete[] arr[i];
	}
	delete[] arr;
}
